Tentang Alynspire & Keamanan

Keamanan & Privasi

  • HTTPS / SSL wajib
  • Hash password (bcrypt), 2FA untuk akun sensitif
  • Validasi input, proteksi CSRF & XSS
  • Pembayaran melalui payment gateway resmi (Midtrans / Xendit) ‚Äî kami tidak menyimpan data kartu

Integrasi teknis (catatan developer)

  1. Autentikasi: OAuth2 + email/OTP. Simpan session dengan secure, httpOnly cookie.
  2. Realtime chat: WebSocket atau Firebase Realtime/Firestore + security rules.
  3. Payments: panggil backend untuk generate token/payment intent dari Midtrans/Xendit.
  4. Backup: otomatis ke cloud (AWS/GCP) & enkripsi at rest.
